属性

图标属性分静态属性和实例属性，静态属性会应用到所有图表，实例属性仅对当前图表有效，新创建一个图表实例，其属性默认为静态属性。

实例属性

属性 取值范围 默认值 读写类型 说明
id 任意字符串 null 只读 图表id, 页面加载后，以此id为标示的页面元素会被替换为图表对象，id保留。页面元素通常是div标签，也可换作其他。
width 数字(500)或者百分比('100%') 400 读写 图表宽度
height 数字(500)或者百分比('100%') 300 读写 图表高度
style white/black white 只读 图表样式，决定图表整体视觉风格
configXML XML格式的字符串 null 只读 图表配置文件
dataXML XML格式的字符串 null 只读 图表数据文件

静态属性

静态属性从全局对象FiCharts上设置，图表默认宽高，样式，数据加载信息提示都以静态属性方式设置。每新创建一个图表实例，静态属性作为默认值赋给对应实例属性。

属性 取值范围 默认值 读写类型 说明
width 数字(500)或者百分比('80%') 400 读写 图表宽度
height 数字(500)或者百分比('80%') 300 读写 图表高度
style white/black white 读写 图表样式，决定图表整体视觉风格
message.noData 任意字符串 暂无数据 读写 未加载数据时的信息提示
message.loadingData 任意字符串 数据加载中... 读写 加载数据时的信息提示
message.loadingDataError 任意字符串 数据加载失败 读写 数据加载失败时的信息提示
