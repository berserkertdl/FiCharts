图表类型与序列

创建图表的类有Chart2D和Pie2D两个。Pie2D仅用来构建2D饼状图；其他图表均由Chart2D构建，配置文件决定具体的图表类型。 


			

			

				这句代码定义了一个柱状图序列，注意节点名
				column
				代表序列类型，除此之外可以用到的序列明如下：
			

			

				

					
						

							标签
							所代表序列类型
						

						

							line
							趋势图
						

						

							column
							柱状图
						

						

							bar
							条形图
						

						

							area
							区域图
						

						

							marker
							散点图
						

						

							bubble
							气泡图
						

						

							stackedColumn
							堆积柱状图
						

						

							stackedPercentColumn
							百分百堆积柱状图
						

						

							stackedBar
							堆积条形图
						

						

							stackedPercentBar
							百分百堆积条形图
						

						

							stack
							堆积图中的一个子序列
						

					

				

			


			

				这些是迄今为止ficharts可以创建出的图表序列类型, 也代表了可创建的图表种类，千变万化的图表类型其实就是这些不同序列的有机组合。一个
				column
				序列构建的是柱状图，两个以上则是多维柱状图，再加一个
				line
				就成了混合图表。
					所有这些序列有着公共属性，也有特有属性。
			

			
序列共有属性
 
			

				

				

					属性名称
					参数类型
					取值范围
					描述
				

				

					xField
					String
					任意字符串
					
						序列的x数据字段，一条数据记录中包含多个字段，这些数据可被多个序列采用，但单个序列通常仅需要两个字段的数据：x,y字段，
						xField
						定义了每条数据记录中用于此序列的x字段
					
				

				

					yField
					String
					任意字符串
					
						序列的y数据字段，一条数据记录中包含多个字段，但序列通常仅需要两个字段的数据：x,y字段，
						yField
						定义了每条数据记录中用于此序列的y字段
					
				

				

					colorField
					String
					
						null
						/任意字符串
					
					
						序列的颜色数据字段，与属性
						color
						不同的是颜色字段会影响到序列中每个数据节点的颜色，序列会根据颜色字段从数据记录中提取每个节点的颜色。默认为
						null
						，系统自动分配节点颜色
					
				

				

					seriesName
					String
					
						null
						/任意字符串
					
					序列名称，多序列或者混合图表时用以区分不同序列。会在图例，工具提示中用到。
				

				

					color
					String
					
						null/hex颜色值，例如
						#FF0000
						，颜色值中的
						#
						可省略
					
					序列颜色，用来区分不同序列。默认系统会对每条序列自动分配颜色，如果要强调某条序列可以通过此属性设置序列专有颜色。
				

				

					xAxis
					String
					null/任意字符
					序列x轴的id, 默认系统自动分配坐标轴，当定义多坐标轴时需要制定序列以那个轴为自己的x轴。
				

				

					yAxis
					String
					null/任意字符
					序列y轴的id, 默认系统自动分配坐标轴，当定义多坐标轴时需要制定序列以那个轴为自己的y轴。
				

				

					singleColor
					String
					true/false
					
						单个序列下，序列中节点是否采用相同颜色，默认
						true
						, 整个序列的颜色都是统一的。例如在柱状图中如果设置为
						false
						时每个柱体颜色均由系统分配。但在多序列时此属性无效，每条序列颜色独立统一。
					
				

				

					labelDisplay
					String
					normal/vertical/inner/none
					
						数值显示方式，none为不显示，vertical为垂直方式显示。不同序列默认参数不同，对于柱状图和条形图normal表示在柱体外部显示，innner表示在柱体内部显示。
					
				

			

			

			
			

				除了这些共有属性外，有些序列有着自身专有属性，本章节中仅涉及常用到的属性，序列详细的样式XML API后继章节会另作阐述。
			


			

				line/area
				序列特有属性
			

			

				

					

						属性名称
						参数类型
						取值范围
						描述
					

					

						smooth
						String
						false/true
						
							默认趋势图以折线方式绘制，此属性设置为
							true
							后折线会变成光滑曲线。
						
					

					

						step
						String
						false/true
						
							默认趋势图以折线方式绘制，此属性设置为
							true
							后折线会变成阶梯线，优先级高于
							smooth
							。
						
					

				

			


			

				column/bar
				序列特有属性
			

			

				

					

						属性名称
						参数类型
						取值范围
						描述
					

					

						maxItemSize
						Number
						正数
						默认100像素, 单个柱体的最大宽度/单个条形图的最大高度
					

				

			

			
			

				bubble
				序列特有属性
			

			

				

				

					属性名称
					参数类型
					取值范围
					描述
				

				

					bubbleField
					String
					任意字符串
					此字段值会影响到气泡的大小
				

				

					maxRadius
					Number
					
						正数, 大于
						minRadius
					
					气泡半径的最大值, 默认为50
				

				

					minRadius
					Number
					
						正数,小于
						maxRadius
					
					气泡半径的最小值, 默认为5
