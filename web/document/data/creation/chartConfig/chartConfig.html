配置文件为XML格式，以config为根节点，其下包含坐标轴axis，序列series和非必须的数据data节点。为了让图表配置简单，data节点常被包含在配置文件中，这样仅需加载一份配置文件便可渲染出图表。配置文件也可与数据分离，动态更新数据或复用配置与数据文件。配置文件根节点上通常可以设置标题与副标题，除此以外还可设置其他特性：

属性 取值范围 默认值 说明
title 任意字符 为空 主标题
subTitle 任意字符 为空 副标题
animation true/false true 初始动画，默认第一次渲染时有动画效果
fullScreen true/false true 开启后双击图表空白区域会全屏显示，再次双击或按下Esc键后推出退出全屏模式
xSuffix 任意字符 为空 x值的后缀
ySuffix 任意字符 为空 y值的后缀
zSuffix 任意字符 为空 z值的后缀, 堆积/气泡图中才会用到z
xPrefix 任意字符 为空 x值的前缀
yPrefix 任意字符 为空 y值的前缀
zPrefix 任意字符 为空 z值的前缀, 堆积/气泡图中才会用到z
precision 非负整数 0 保留小数点位数，仅对数字类型数据有效
useGrouping true/false false 是否群组方式显示数据，如数字325,682设置了useGrouping为true后显示为325,682，此特性在显示货币类型数据时会用到。
bgVisible true/false true 是否隐藏图表背景

配置接口与图表数据下一节会详细介绍，序列与坐标轴部分请参见相关章节。
