Flash/AIR项目中的应用

ficharts不仅可以用于web项目,还可借助于AIR技术应用在桌面/移动设备中。安装包内的FiCharts.swc就是专门针对于Flash/AIR型应用的类库。通过实例化Chart2D, Pie2D两个类可构建多种图表。其中Pie2D仅用来构建2D饼状图，其他类型的图表由Chart2D类构建(通过配置文件定义图表类型)，他们有着极其类似的接口：

接口/属性 参数 默认值 说明 适用于
setConfigXML(xml) XML格式的字符串 为空 图表的配置文件 Chart2D, Pie2D
setDataXML(xml) XML格式的字符串 为空 图表的数据文件 Chart2D, Pie2D
setConfigFileURL(url) 字符串 为空 配置文件的路径 Chart2D, Pie2D
setDataFileURL(url) 字符串 为空 数据文件的路径 Chart2D, Pie2D
setStyle(name) white/black white 图表样式 Chart2D, Pie2D
render() 无 无 渲染图表，对图表做了任何属性设置后都需调用此方法才能看到结果 Chart2D, Pie2D
width 非负数 150 图表宽度 Chart2D, Pie2D
height 非负数 150 图表高度 Chart2D, Pie2D
